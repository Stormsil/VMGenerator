Для добавления иконки приложения:

1. Создайте или найдите иконку для приложения в формате .ico
2. Сохраните её как "app.ico" в папке Assets/
3. Иконка должна содержать несколько размеров: 16x16, 32x32, 48x48, 256x256

Онлайн-конвертеры ICO:
- https://www.icoconverter.com/
- https://convertio.co/ru/png-ico/
- https://www.favicon-generator.org/

Или используйте готовый шаблон Proxmox-стиля:
- Тёмный фон #1a1a1a
- Символ VM или Server
- Акцентный цвет #5c9fd6
